## Sample Prompt Used to Test the Agent

The following prompt was used to evaluate the agent’s ability to process a high-pressure, enterprise-style threat hunting scenario. It was designed to simulate a real-world Falcon alert escalation involving suspected advanced persistent threat activity in a financial services environment.

This sample tests whether the agent can:
- extract indicators of compromise from mixed security artifacts
- reason over operational context
- map activity to MITRE ATT&CK
- generate hunting queries for multiple platforms
- support incident response under time pressure
